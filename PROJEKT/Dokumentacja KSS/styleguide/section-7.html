7 #7 ZAPIS

Toggle full screen Open in new window Toggle example guides Toggle HTML markup

Formularz który posiada walidację i realizuje zapisanie się na dany kurs. Posiada:
- pięć pól tekstowych,w tym dwa wymagające wpisania danych w konretnym formacie.
- dwa selecty zawierający opcje wyboru języka oraz jego poziomu.
- jeden checkbox z akceptacją regulaminu.
- przycisk wysyłający formularz po przejściu walidacji.
- wszystkie pola formularza są wymagane.
Example

Zapisz się

Markup 
<article id="register">
                <section class="st">
                    <h2>Zapisz się</h2>
                </section>
                <div class="nd">
                    <div class="formcontrol">
                        <form>
                            <label for="name">Imie:</label>
                            <input type="text" id="name" pattern="[A-Za-zĄąĆćĘęŁłŃńÓóŚśŹźŻż]+" maxlength="20" required>
                            <label for="surname">Nazwisko:</label>
                            <input type="text" id="surname" pattern="[A-Za-zĄąĆćĘęŁłŃńÓóŚśŹźŻż]+" maxlength="20" required>
                            <label for="email">Email:</label>
                            <input type="text" id="email" pattern="[a-z0-9._%+\-]+@[a-z0-9.\-]+\.[a-z]{2,}$" title="Email musi mieć format example@example.com!" required>
                            <label for="adres">Adres zamieszkania:</label>
                            <input type="text" id="adres" pattern="[A-Za-zĄĆĘŁŃÓŚŹŻąćęłńóśźż0-9\s]*" required>
                            <label for="kodPocztowy">Kod pocztowy:</label>
                            <input type="text" id="kodPocztowy" pattern="\d{2}-\d{3}"  title="Kod pocztowy musi mieć format XX-XXX, gdzie X to cyfra!" required>
                            <label for="miejscowosc">Miejscowość:</label>
                            <input type="text" id="miejscowosc" pattern="[A-Za-zĄąĆćĘęŁłŃńÓóŚśŹźŻż]+" required>
                            <label for="jezyk">Wybierz język:</label>
                            <select id="jezyk">
                                <option>Angielski</option>
                                <option>Niemiecki</option>
                                <option>Hiszpański</option>
                                <option>Włoski</option>
                                <option>Francuski</option>
                            </select>
                            <label for="poziom">Wybierz poziom:</label>
                            <select id="poziom">
                                <option>A1</option>
                                <option>B2</option>
                                <option>C1</option>
                                <option>C2</option>
                            </select>
                            <label for="regulamin">Akceptuje regulamin szkoły: </label>
                            <input type="checkbox" id="regulamin" required>
                            <button type="submit">Wyślij</button>
                        </form>
                    </div>
                </div>
            </article>

7.1 #7.1 KONTAKT

Toggle full screen Open in new window Toggle example guides Toggle HTML markup

Formularz posiadający walidację dzięki ktoremu można się skontaktować ze szkołą. Posiada dwa pola tekstowe oraz textarea do wpisania treści wiadomosci.
Example

Napisz do nas!

Markup 
<article id="kontakt">
            <section class="st">
                <h2>Napisz do nas!</h2>
            </section>
            <div class="nd">
                <div class="formcontrol">
                    <form>
                        <label for="name">Imie:</label>
                        <input type="text" id="name" pattern="[A-Za-zĄąĆćĘęŁłŃńÓóŚśŹźŻż]+" maxlength="20" required>
                        <label for="surname">Nazwisko:</label>
                        <input type="text" id="surname" pattern="[A-Za-zĄąĆćĘęŁłŃńÓóŚśŹźŻż]+" maxlength="20" required>
                        <label for="tresc">Treść:</label>
                        <textarea></textarea>
                        <button type="submit">Wyślij</button>
                    </form>
                </div>
            </div>
        </article>
